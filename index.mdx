## What is Neru?

**Neru (練る)** — Japanese for _"to refine and master through practice"_ — is a keyboard-driven navigation tool for macOS that lets you control your entire desktop without touching your mouse or trackpad.

It's the free, open-source alternative to Homerow, Mouseless, and Wooshy — with zero paywalls, zero subscriptions, and everything configurable down to the last pixel.

## Quick Start

<Steps>
  <Step title="Install Neru">
    <CodeGroup>
    ```bash Homebrew
    brew tap y3owk1n/tap
    brew install --cask y3owk1n/tap/neru
    ```
    ```bash Nix
    # inputs.neru.url = "github:y3owk1n/neru";
    # See installation guide for nix-darwin / home-manager setup
    ```
    ```bash From Source
    git clone https://github.com/y3owk1n/neru.git
    cd neru && just release
    ```
    </CodeGroup>
  </Step>
  <Step title="Grant Accessibility Permission">
    Open **System Settings → Privacy & Security → Accessibility** and enable Neru.
  </Step>
  <Step title="Launch and Try">
    ```bash
    # Start Neru
    open -a Neru

    # Try the default hotkeys
    # Cmd+Shift+C - Recursive grid mode
    # Cmd+Shift+Space - Hints mode
    # Cmd+Shift+G - Grid mode
    # Cmd+Shift+S - Scroll mode
    ```
  </Step>
</Steps>

## Navigation Modes

Neru provides three powerful navigation modes that work together:

<CardGroup cols={3}>
  <Card title="Recursive Grid" icon="grid-2" href="/guides/recursive-grid">
    Divide your screen into a coordinate grid and drill into any region with u/i/j/k until you land exactly where you want.
  </Card>
  <Card title="Grid Mode" icon="table-cells" href="/guides/grid-mode">
    Coordinate-based navigation without recursive subdivision, great for quick coarse navigation.
  </Card>
  <Card title="Hints Mode" icon="tags" href="/guides/hints-mode">
    Labels appear on every clickable element for instant accessibility-driven navigation.
  </Card>
</CardGroup>
